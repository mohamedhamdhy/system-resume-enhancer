## 🚦 Getting Started

### Prerequisites

- Node.js >= 18.x
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/mohamedhamdhy/micro-ts-resume-sys
cd micro-ts-resume-sys
```

2. **Install dependencies**
```bash
npm install
```

3. **Run the application**
```bash
# Development mode
npm run dev
# Production mode
npm start
```

The server will start on `http://localhost:3000` (or your configured port).

## 🔍 Troubleshooting

**JWT Token Invalid**
- Verify `JWT_SECRET` is set in `.env`
- Check token expiration
- Ensure clock synchronization between services
